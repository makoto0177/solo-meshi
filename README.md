## 概要

「ソロメシ」は、一人での外食に特化した飲食店共有プラットフォームです。
一般的なグルメサイトでは店舗詳細に埋もれている「一人で入りやすさ」に関する情報を、一覧で一目で比較できるようにし、実際に訪れたユーザーの体験を気軽に共有・参照できます。

## 開発背景

私は普段、一人で出かけることが多く、その際に外食する機会も多いのですが、「一人で入りやすい飲食店を探すこと」が意外と難しいと感じていました。

知人や友人に良い店を聞くこともできますが、以下の課題がありました：
- 「自分のためだけ」に聞くのは心理的ハードルが高い
- 相手からのリアルタイムでの返信は難しい

また、既存のグルメサイトにも「一人で入りやすい」という情報やカウンター席の有無は掲載されていますが、店舗詳細ページに埋もれており、複数の店を比較する際に何度も詳細を確認する手間がかかります。

結果として、**「一人で入りやすい店を知りたいのに、気軽に情報を得られない」** という不便さを解決するために、このアプリを開発することにしました。

## アプリの目的

一人で入りやすい飲食店を簡単に探せて、自分の体験を気軽に共有できる Web アプリを提供すること。

一般的なグルメサイトでは店舗詳細に埋もれている以下の情報を、"おひとりさま視点"で一覧表示し、簡単に比較できるようにする：
- 一人で入りやすさ
- 店の雰囲気（一人客の多さ、カジュアル度、静かさ）
- カウンター席の有無（多い/少ない）
- 実際の混雑度・入りやすさのリアル情報

これにより、ユーザーが安心して新しい店を選べる状態を作る。

また、投稿機能により、ユーザーは他の人の役に立つだけでなく、自分自身の「一人飯ログ」として活用でき、過去に訪れた良い店を振り返ることができる。

## 主な機能

### 1. 飲食店検索・一覧機能
- エリアやキーワードで一人で入りやすい飲食店を検索
- 一覧画面で「一人での入りやすさ」「カウンター席の有無」「一人客の多さ」を一目で比較
- 構造化されたデータで効率的な店選びが可能

### 2. 体験投稿・共有機能
- 実際に訪れた飲食店の情報を気軽に投稿
- 一人での入りやすさ（5段階評価）
- カウンター席の有無・規模感（なし/少ない/普通/多い）
- 店の雰囲気（一人客の多さ、カジュアル度、静かさ）
- コメントと写真のアップロード

### 3. 詳細情報の閲覧
- 他のユーザーの投稿を参考に店選びができる
- リアルな体験談から店の雰囲気を把握
- 同じ「おひとりさま」視点での情報のみが集約

### 4. マイページ機能（一人飯ログ）
- 自分の投稿一覧の管理
- 「一人飯ログ」として個人的な記録に活用
- 過去に訪れた良い店を思い出したい時に確認
- 投稿の編集・削除

## 既存サービスとの差別化

食べログなど既存のグルメサイトでも一人で入りやすい店は探せますが、大きく2つの差別化ポイントがあると考えています。

### 1. 情報の特化と見やすさ
既存サイトでは総合的なレビューの中に一人飯情報が埋もれていますが、ソロメシでは一覧画面で「一人での入りやすさ」「カウンター席の有無」「一人客の多さ」といった情報を構造化して一目で比較できるようにしています。

また、既存サイトでは「ひとり」というキーワードを機械的に検索しているため、「店員が一人で対応していた」「一人で訪れた」といった無関係な情報も混在していますが、ソロメシでは全ての投稿が一人客視点に特化しています。

### 2. 投稿のハードルの低さ
食べログでは総合的なレビュー（料理、サービス、雰囲気など）を書く必要がありますが、ソロメシでは一人飯視点だけに絞ることで、気軽に体験を共有できる設計にしています。

## ターゲットユーザー

- 一人で外食することが多い社会人
- 新しい店に入りたいが「一人で浮かないか」を気にする人
- 仕事帰りにサクッと一人ご飯したい層
- チェーン店以外も開拓したいが、店の雰囲気が分からず不安な人

## 利用シナリオ

### シナリオ1：出かけた先で一人ご飯したい
1. ユーザーは「新宿」「渋谷」などのエリアを検索
2. 一覧で「一人で入りやすさ（★平均）」やコメントを確認
3. 詳細ページから雰囲気や注意点を見て店を決める

### シナリオ2：行って良かった店を共有したい
1. 一人で行った店が良かった
2. 店を新規投稿（店名 / エリア / カテゴリ / 入りやすさ / コメント）
3. 他ユーザーがその情報を参考にできる

### シナリオ3：自分用メモとして使いたい
1. 行った店を投稿しておくことで、自分の「一人飯ログ」として蓄積
2. マイページで自分の投稿一覧を見返せる
3. 過去に訪れた良い店を思い出したい時に記録から確認できる
